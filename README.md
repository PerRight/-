# Hi-Flow 소프트웨어 — 3D 히트맵 & 실시간 운영 대시보드

수중드론(Hi-Flow)이 수면을 이동하며 측정 지점마다 센서를 수심별로 하강시켜
수집한 수질 데이터를 다루는 파이썬 프로젝트.

1. **누적 데이터 3D 히트맵** — 위치(X, Y)와 수심(Z)을 통합한 시각화
2. **실시간 운영 대시보드** — 운영자가 플랫폼 상태를 실시간으로 확인하는 화면

## 실시간 운영 대시보드

```bash
python dashboard_server.py   # → 브라우저에서 http://localhost:8000 접속
```

추가 설치 불필요(표준 라이브러리만 사용). 단, 3D 히트맵은 Plotly CDN을 사용하므로
인터넷 연결이 필요하다. 1초 주기로 자동 갱신되며 표시 항목:

| 항목 | 표시 방식 |
|---|---|
| GPS 위치 | 위도·경도 + 수역 내 이동 궤적 미니맵 (○ = 다음 측정 지점) |
| 플랫폼 상태 | 이동 중 / 정지·안정화 중 / 측정 중 / 센서 회수 중 / 지점 측정 완료 |
| 센서 연결 상태 | 센서별 연결됨 / ⚠ 끊김 배지 |
| 현재 측정 깊이 | 수치 + 수심 게이지 (0.5 / 1.0 / 1.5 m — 수심당 1분, 지점당 3분) |
| pH, EC | 현재값 + 최근 90초 추이 스파크라인 |
| 측정 시간 | 최근 측정값 수신 시각 + 가동 시간 |
| 배터리 상태 | 잔량 % + 정상 / 주의 / 위험 상태 표시 |
| 측정 완료 여부 | 현재 지점 완료 배지 + 전체 진행률 (완료 지점 / 121) |
| **실시간 3D 히트맵** | 측정이 누적될 때마다 채워지는 3D 히트맵 — 2초마다 자동 갱신, 드래그로 회전(시점 유지), 표시 항목(pH/EC/…) 선택 가능. 보기 형태 2종: **막대**(지점별 수심 평균을 높이로 — 프로토타입 예시 스타일, 기본값) / **셀**(수심별 상세). 탐사 회차(실시간/전체 누적/회차별) 선택, 임계값 초과 오염 셀은 빨간색 표시, Excel 내보내기 버튼 제공 |

### 센서 확장 방법 (DO, 탁도 등)

`dashboard_server.py` 상단의 `SENSORS` 목록에 항목 하나만 추가하면
대시보드에 센서 카드가 자동 생성된다 (파일에 주석 예시 있음):

```python
{
    "id": "do", "name": "DO 용존산소", "unit": "mg/L", "decimals": 2,
    "read": lambda d, rng: 8.5 - 0.5 * d + rng.gauss(0, 0.1),
},
```

실제 드론 연동 시에는 시뮬레이터 스레드 대신 수신 데이터로
`TELEMETRY` 딕셔너리를 갱신하면 화면은 수정 없이 그대로 동작한다.

## 누적 데이터 3D 히트맵

### 실행 방법

```bash
pip install -r requirements.txt

# 1. 누적 측정 데이터 생성 (시뮬레이션) → data/measurements.csv
python generate_data.py

# 2. 3D 히트맵 렌더링 → output/heatmap_3d.html, output/heatmap_3d.png
python visualize_heatmap.py
```

### 결과물

| 파일 | 용도 |
|---|---|
| `output/heatmap_3d.html` | 인터랙티브 3D 히트맵 — 탐사 회차·센서 드롭다운, 마우스로 회전·확대, 셀에 올리면 위치·수심·평균값·누적 측정 횟수 표시 |
| `output/heatmap_3d_ph.png`, `output/heatmap_3d_ec.png` | 정적 3D 히트맵 (전체 누적, 센서별) — 보고서·발표 자료용 |
| `output/report.xlsx` | 수치 요약 보고서 (`python export_report.py`) — 회차별 요약 / 오염 셀 목록 / 셀별 전체 데이터 |

### 데이터 형식

`data/measurements.csv` — 드론 실측 데이터를 이 형식으로 넣으면 시뮬레이션 없이 그대로 시각화 가능.

| 컬럼 | 설명 |
|---|---|
| `run_id` | 측정 회차 |
| `timestamp` | 측정 시각 |
| `x_m`, `y_m` | 측정 지점 평면 좌표 (m) |
| `depth_m` | 측정 수심 (m) |
| `ph` | pH (6.5~8.5 벗어나면 오염) |
| `ec` | EC 전기전도도 (µS/cm, 500 초과 시 오염) |

### 시각화 설계

- 누적 측정값을 (X, Y, 수심) 격자 셀별 **평균**으로 집계 — 회차가 쌓일수록 셀당 표본이 늘어 정밀해짐
- 색상은 단일 색상(파랑) 순차 램프: 밝음 = 깨끗함 → 어두움 = 오염도 높음
  (무지개 컬러맵은 색약자 구분·명도 순서 문제로 사용하지 않음)
- 탁도가 높은 셀일수록 마커를 크게 그려 낮은 값 셀에 가려지지 않고 내부 핫스팟이 보이도록 함
- 수심 축은 아래 방향이 깊어지도록 반전
- 오염 판정 임계값(pH 6.5~8.5 / EC 500 µS/cm)은 `thresholds.py` 한 곳에서 관리 —
  임계값을 넘는 셀은 빨간 다이아몬드로 표시
